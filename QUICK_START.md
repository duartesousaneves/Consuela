# Consuela — Início Rápido

## Não há instalação. Não há configuração.

As ferramentas Gmail e Google Calendar estão ligadas via MCP ao Claude Code e ao projeto Consuela no claude.ai. Basta falar.

---

## Opção 1 — Terminal (Claude Code)

Abre o Claude Code (terminal Antigravity) e escreve o que precisas:

```
"mostra-me os emails por ler"
"que tenho no calendário amanhã?"
"cria uma reunião segunda às 10h com o título Revisão Semanal"
```

---

## Opção 2 — Browser / App

1. Abre [claude.ai](https://claude.ai)
2. Vai a **Projects → Consuela**
3. Escreve o que precisas

Funciona igual no telemóvel via app Claude.

---

## Exemplos de Pedidos

### Email
- "Quantos emails não li?"
- "Resume os emails não lidos de hoje"
- "Procura emails do João sobre o projeto X"
- "Cria um rascunho a responder que aceito a reunião"

### Calendário
- "O que tenho esta semana?"
- "Tenho alguma coisa quinta de manhã?"
- "Cria um evento Dentista na sexta às 9h"
- "Aceita o convite da reunião de amanhã"

---

## Custo

Zero adicional. Usa a subscrição claude.ai existente.
